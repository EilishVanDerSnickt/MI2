Omschrijving project MI2

Ik zou een variant van het gezelschapspel galgje willen maken. Zodat een random generator een woord genereert en de gebruiker via een tekstvak letters kan raden, elke keer als de gebruiker het fout heeft wordt er een stukje van het mannetje getekend. Als het mannetje helemaal getekend is verlies je. Elke keer dat een gebruiker een letter raadt dan worden deze letter(s) getoond op de juiste plaats in het woord. Als je het hele woord kan raden voor het mannetje helemaal getekend is win je het spel.
